Candidate 04

Deepen Desktop Document Workflows

Worth exploringlocal-substitutable
lib/nodepad-format.ts · lib/export.ts · app/page.tsx · next.config.mjs · app/layout.tsx · package.json

Problem

Archive behavior and browser mechanics are fused, validation is shallow, and the current runtime has no native backup or restore path.

Solution

Deepen only document workflows with real behavior; keep native dialogs, filesystem access, and browser downloads as adapters rather than creating a generic platform pass-through.

  • Locality: versions migrate once
  • Leverage: archive + backup
  • Native dialogs stay internal
  • Malformed data fails closed
Caution: a generic “platform module” fails the deletion test when it only renames Tauri calls. Deepen workflows with policy, not pass-throughs.
