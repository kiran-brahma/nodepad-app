# Triage Labels

The engineering skills use five canonical triage roles. This table maps each role to the exact GitHub label used by this repository.

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer must evaluate the issue |
| `needs-info` | `needs-info` | Waiting for information from the reporter |
| `ready-for-agent` | `ready-for-agent` | Decision-complete and ready for an AFK agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or intervention |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a triage role, use the corresponding GitHub label from this table.
